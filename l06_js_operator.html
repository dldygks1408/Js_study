산술 연산자

operator(연산) 이름 설명 예시 JS
+ 더하기 두 수를 더한다. 6.8 + 4.2
- 빼기 두 수를 왼쪽 기준으로 뺀다. 10.5 - 30
* 곱하기 두 수를 곱한다. 10.3 * 20
/ 분할 왼쪽 수를 오른쪽 수로 나눈다. 20 / 4
% 나머지 나누고 남은 나머지 값 10%3
a**b 거듭제곱 자신의 수 a를 b만큼 곱한다. 2**5
Math.pow(2,5)

할당 연산자와 증가 및 감소 연산자

operator(연산) 이름 설명 예시
a += b 추가할당 a = a + b : 자기 자신 a에 b을 더한다. x += 10 ( x= x + 10 )
a -= b 빼기할당 a = a - b : 자기 자신 a에 b을 뺀다. x -= 10 ( x= x - 10 )
a *= b 곱셈할당 a = a * b : 자기 자신 a에 b을 곱한다. x *= 10 (x = x * 10)
a /= b 분할할당 a = a / b : 자기 자신 a에 b을 나눈다. x /= 10 ( x = x / 10 )
++a; or --a 증가,감소 연산자 (우선증가) a=a+1; or a=a-1;
숫자 앞에 증가 및 감소 연산자가 오면 연산을 먼저 하고 다른 실행을 옮긴다.		 a(++i) : i를 증가시키고 a()를 실행
a++; or a-- 증가,감소 연산자 a=a+1; or a=a-1;
숫자 뒤에 증가 및 감소 연산자가 오면 다른 실행을 하고 연산 한다.		 a(i++) : a()를 실행 시키고 i를 증가

비교 연산자

참과 거짓을 테스트 하는 비교 연산자

operator(연산) 이름 설명 예시
a == b 동등 a 와 b의 값이 같은지 비교 10 == 10 :
a != b 비동등 a 와 b의 값이 같지 않은지 비교 10 != (6+4)
a === b 엄격한 동등 a 와 b 가 완전히 동일해야한다. 10 === "10"
a !== b 엄격한 비동등 a 와 b 가 완전히 동일하지 않은지 비교. 10 === "10"
a < b 미만(작다) 왼쪽 값 a가 오른쪽 값 b 보다 작은지 확인 5 < 3 : false
a > b 초과(크다) 왼쪽 값 a가 오른쪽 값 b 보다 큰지 확인 5 > 3 : true
a <= b 이하 왼쪽 값 a가 오른쪽 값 b 보다 같거나 작은지 확인 5 <= 3 : false
a >= b 이상 왼쪽 값 a가 오른쪽 값 b 보다 크거나 같은지 확인 5 >= 3 : true
