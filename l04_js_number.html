숫자와 부동 소수점을 포함하는 Number type

number는 64비트의 부동 소수점인 숫자를 저장하는 기본데이터 타입

  • number가 표현 할 수 있는 최대 수 (MAX_FAFE_INTEGER)
  • number가 표현 할 수 있는 최소 수 (MIN_SAFE_INTEGER)
  • 자바스크립트가 표현 할 수 있는 가장 큰 수 (MAX_VALUE)
  • 자바스크립트가 표현 할 수 있는 가장 낮은 수(MIN_VALUE)
  • Number. 는 정수와 소수를 표현한 데이터 타입을 자바스크립트가 브라우저 동작시 생성한 객체
  • Number는 총 64bit를 저장하고 분수는 0~51, 지수는 52~62, 부호(.)이 64에 저장된다.
  • 안전하지 않은 이유 0.2 + 0.1 =
  • 안전하지 않은 이유 9999999999999999 =
  • let eNum=567e7 eNum=567e-7
