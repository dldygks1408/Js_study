jQuery 라이브러리

  • $(selectorStr) : 제이쿼리 선택자 ( == document.querySelectAll(selectorStr) )
  • $(".jq_ul li").click( function(props){ ... } ) == document.querySelectAll(".jq_ul li").foreach(li=>{ li.onclick = function(props){ ... } })
  • 제이쿼리는 이벤트를 chain 할 수 있다.
  • 제이쿼리는 복수의 선택자로 찾은 dom을 반복문 없이 재정의할 수 있다. ( 반복문 생략 )
  • 제이쿼리를 시작하려면 꼭 $()를 작성해야한다.
