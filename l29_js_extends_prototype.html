프로토타입의 상속

  • 프로토 타입은 함수와 객체가 생성될때 만들어진다.
  • 프로토타입의 상속은 __proto__로 프로토타입을 연결하는 것
  • 원시적 상속 : c_obj.__proto__ = f_obj
  • create 상속 : c_obj = Object create(f_obj) (결과는 위와 같다.)
  • 인스턴스 객체 생성 (new F()) : new F().__proto__ = F(){}, new F().__proto__.__proto__ = F.prototype
  • class A extends B{} : A.__proto__ = B.prototype
  • B = Object.assign(A) : A가 B 의 prototype을 공유 한다. (A의 필드를 B에 복사)
  • 생성된 프로토타입은 하나가 공유된다. 때문에 프로토타입이 바뀌면 그것을 상속하는 자식 또한 바뀐다.
