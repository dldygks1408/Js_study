css float로 블럭 정렬하기

float는 글자가 여러줄 있을때 그 사이 왼쪽이나 오른쪽에 블럭을 삽입하기 위해 존재한다.

display : inline-block은 글자가 한줄 있을때 삽입하기 유용하다.
(블럭의 높이와 상관없이 한줄에 삽입되기 때문)

inline-block
34세 직장인 A씨는 2년 전부터 비상장 스타트업에 1억원가량을 투자하고 있다. 한 벤처 액셀러레이터(AC)를 통해 개인투자조합에 5000만원을 가입하고 화장품 원료를 생산하는 바이오 스타트업에 1000만원, 프롭테크 기업에는 2000만원을 투자했다.
float : right
그 밖에 개별적으로 300만~500만원씩 초기 스타트업에 넣었다. 한 자산운용사에서 벤처투자 업무를 맡은 지인이 투자금을 넣을 때 같이 투자하기도 한다. A씨가 투자한 업체는 10여 곳. 그는 “투자 기업 중 한 곳만 터져도 20~100배가량의 ‘잭팟’이 가능하다”며 “매일 차트를 들여다보며 마음을 졸이고 투자 포트폴리오를 조정해야 하는 일반 상장주식보다 돈을 묶어놓고 3~7년 뒤 회수하는 방식이 오히려 마음 편하다”고 했다.

문제 해결 1 clear

  • 형제 노드가 float를 컨텐츠로 인지 못하는 문제 해결 (clear : left, right, both)
    • float : left
    container

    문제 해결 2 overflow

  • 부모 노드에 overflow를 visible(default) 외의 값을 준다.(overflow : auto, hidden ...)
    • float : left
    container

    문제 해결 3 display : inline-block

  • 부모 노드가 inline-block이면 자식이 float인 것을 컨텐츠로 인지한다.
    • float : left
    container

    문제 해결 4 .clearfix::after

  • .clearfix::after{ content : ""; clear : both; display : table; }
    • float : left
    container
