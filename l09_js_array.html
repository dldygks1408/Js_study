자바스크립트의 배열

여러 변수를 index로 참조해 저장하는 데이터 타입(Iterator 상속하고 있다.)

작성법 1 : const array = [item1, item2, item3, ...] (기본데이터타입 처럼 사용 가능 권장)

작성법 2 : const array = new Array(item1, item2, item3, ...) (권장하지 않음)

호출법 : array[index]

["a", "b", "c", "d"].toString() : ( 객체를 바로 호출하면 toString() 생략되어있다. )

대 입 : array[index] = value 예)a[1]="B"

typeof [] : Object / 배열의 타입을 typeof로 검사 할 수 없다. Array.isArray(a) 배열이면 true, 아니면 false

Array[maxIndex+1] = value : (자바스크립트는 동적으로 배열의 길이를 조절한다.)
