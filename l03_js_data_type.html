자바스크립트의 데이터 타입과 기본 데이터 타입

자바스크립트의 변수는 모든 데이터 타입을 참조 할 수 있다.(동적타입 dynamically typed)
이 때 모든 프로그래밍 언어와 같이 자주 쓰이는 타입을
기본데이터 타입으로 지정하고 있으며 총 7가지가 있다.
(string, number, bigint, boolean, symbol, null, undefined)

Number 숫자형

정수와 (float) 부동소수점, 무한대(Infinity), NaN(Not a Number) 숫자가 아님의 오류를 나타낸다.
숫자형은 *,+,-,/ 등의 연산을 할 수 있다.

  • 1/0 :
  • -1/0 :
  • "십"/2 :
  • "10"/2 : 자동으로 형 변환한다 ("10" -> 10)
  • typeof : 데이터 타입을 확인 할 수 있다.
  • x = "안녕" :
  • x = 3.14 :
  • x = -1 :
  • x = [1, 2 ,3] :
  • x=new Number(100)
  • 100 == new Number(100) :
  • 100 === new Number(100) 타입까지 확인:
  • 100 기본데이터 타입, new Number(100) 객체 : 둘다 value는 100이고 기본 데이터 타입이 훨씬 가볍다.
  • 객체지향 언어에서 객체는 new 연산자로 생성한다. (자바스크립트는 new를 생략 할 때가 있다.)
    예) Number 클래스로 객체를 생성하기 위해 new Number() 생성자 생성
    기본 데이터 타입은 new 연산자 없이 선언 할 수 있고 매우 가볍다.

기본데이터 타입처럼 사용 할 수 있는 배열과 JSON

let arr = [a, b, c, d]; arr = new Array(a, b, c, d); new 없이 생성 가능 배열타입

배열은 순서를 가지는 데이터타입으로 모든 데이터 타입을 참조 할 수 있다.
(같은 데이터 타입을 참조하는 것을 권장)

let json=[key:value, key2:value] : new 없이 생성 가능한 json 타입

json 은 key 와 value를 가지는 데이터 타입으로 new Object로 생성 가능하다.
이 때 Object는 모든 클래스의 부모인 Object와 별개의 클래스다.

JSON(JavaScript Object Notation) 의 줄인말로 경량 데이터 교환 형식이다.
언어에 독립적으로 문자열 형식이 중요하다.
