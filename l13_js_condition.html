자바스크립트의 조건문

블럭안의 실행을 조건에 따라 동적으로 선택되게 하는 조건문 if, switch, 삼항연산자

if 조건문

if( 조건 ){ 블럭 } : 조건이 참이면(true) 블럭을 실행한다.

if( ){ }else{ } : if의 조건이 맞지 않으면 else의 블럭을 실행한다.

if( 조건1 ){ 블럭 }else if( 조건2 ){ 블럭 }else{ } :
1조건 만족시 1번 블럭 실행 1번 조건이 만족하지 않으면 2번조건 실행, 모두 만족하지 않으면 else 실행

  • if(value < 50) { } :
  • if(value > 20 && value < 50) { } :
  • if(value < 20 || value > 50) { } :
  • if(value>=10 && value<=20 || value>=30 && value<=40){

Switch 조건문

작성법
        switch(표현식 or 변수){
          case a :
            실행
           break;
          case b :
            실행
           break;
          case c :
            실행
           break;
           default:
        }

스위치 케이스는 엄격한 비교(===)을 사용하고 위에서 부터 일치하는 항목을 찾으면 스위치는 종료된다.(엄격한 비교 외의 조건을 사용 할 수 없다.)

if에서 else if 문과 동일하게 작동하지만 조건문의 수행 속도가 굉장히 빠르다.
조건을 파악하기 굉장히 좋다.

삼항연산자

작성법 1 : (조건문) ? true : false == if(조건문){ true }else{ false }

작성법 2 : let variable = (condition) ? trueObject : falseObject;

 작성법 3 :
        (condition) ? true
      : (condition2) ? true
      : (condition3) ? true
      : false;

조건 파악이 편하고 코드가 간결해 프로그래밍 속도를 올려준다.
