자바스크립트 문자열 String의 메소드

문자열은 유저에게 전달하는 데이터로 웹상에서 가장 많이 사용된다. 때문에 문자열을 다루는 함수가 많이 존재 한다.

method 설명 예시
toUpperCase(), toLowercase() 문자열의 대문자 or 소문자로 반환 "aBc deFg".toUpperCase() :
"aBc deFg".toLowerCase() :
trim() 문자열의 앞뒤 공백을 제거한다. " 요요요 ".length :
" 요요요 ".trim().length :
charAt(i), charCodeAt(i) i번째 문자 반환 / i번째 문자 유니코드 반환 "ABCDE".charCodeAt(0) :
"ABCDE".charAt(0) :
"ABCDE"[0] :
slice(start, end) 일부 문자를 추출 할 때 사용 "가가나나다다".slice(2,4) :
"가가나나다다".slice(-4,-2) :
"가가나나다다".slice(4) :
substring(start, end) slice와 동일하지만 양수 인덱스만 사용가능 "가나다라마바".substring(1,2) :
"가나다라마바".substring(-1,-2) :x
substr(start, length) 일부 문자를 추출한다. 시작점부터 길이까지 추출한다. "ABCDE".substr(2,2) :
"ABCDE".substr(-3,2) :
startsWith(string, position) 특정문자열로 시작하는지 여부 (boolean) foodList.startsWith("스") :
endsWith(Sting,position=String.length) 끝에서부터 특정 문자열로 시작하는지 여부 (boolean) foodList.endsWith("위치") :
foodList.endsWith("샌드") :
foodList.endsWith("된장",10) :
includes(String, start position=0) 특정 위치부터 특정 문자열이 있는지 여부를 검색 (boolean) foodList.includes("찌개") :
foodList.includes("찌개",12) :
indexOf() 특정 문자열이 있는지 검색 후 시작 지점을 인덱스로 반환 특정 문자열이 없을때 -1 반환 foodList.indexOf("피자")
foodList.indexOf("피자",6)
lastIndexOf() 끝또는 특정위치에서부터 거꾸로 특정 문자열이 있는지 검색 후 시작 지점(index)을 반환, 특정 문자열이 없을 때 -1을 반환 test.lastIndexOf("Brave") :
test.indexOf("Brave") :
test.lastIndexOf("new",5) :
test.indexOf("new",5) :
split(String, limit length) 특정 문자열을 기준으로 나누어 배열로 만든다. (길이를 정 할 수 있다.) .split("").join("<br>")

replace() 특정 문자열이나 정규식 패턴으로 검색 후 다른 문자열로 변환 값을 반환 (a= a.replace() X) foodList.replace("된장찌개", "스테이크")
replaceAll() 특정 문자열이나 정규식 패턴으로 검색 한 모든 문자열을 다른 문자열로 변환한 값을 반환 foodList.replace("된장찌개", "스테이크")
