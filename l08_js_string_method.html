자바스크립트 문자열 String의 메소드

문자열은 유저에게 전달하는 데이터로 웹상에서 가장 많이 사용된다. 때문에 문자열을 다루는 함수가 많이 존재 한다.

method 설명 예시
toUpperCase(), toLowercase() 문자열의 대문자 or 소문자로 반환
trim() 문자열의 앞뒤 공백을 제거한다. " 요요요 "
" 요요요 "
charAt(), charCodeAt() 0번째 문자 반환 / 0번째 문자 유니코드 반환 "ABCDE".charCodeAt(0) :
"ABCDE".charAt(0) :
"ABCDE"[0] :
slice(start, end) 일부 문자를 추출 할 때 사용 "가가나나다다".slice(2,4) :
"가가나나다다".slice(-4,-2) :
"가가나나다다".slice(4) :
