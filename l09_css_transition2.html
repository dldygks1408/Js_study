transition 이론과 타이밍 함수

  • transition-property : 애니메이션이 가능한 css 속성 (default all)
  • transition-duration : transition이 일어나는 시간. s(초) or ms(밀리초)로 지정 (default 0)
  • transition-delay : transition이 시작하는 시간. s(초) or ms(밀리초)로 지정 (default 0)
  • transition-timing-function : transition 효과를 위한 속도를 함수로 지정 (default ease)
  • https://cubic-bezier.com/ 참고
  • cubic bezier(), ease, liner, ease-in, ease-out, ease-in-out

dataset 개발자가 요소에 직접 속성을 만드는 행위

  • element data- * : element.dataset.* or element.dataset[*]
  • 데이터를 출력할때 요소의 데이터로 노드를 제어하기를 원하면 iuput의 value 처럼 data-key = value 지정 할 수 있다.
  • value의 데이터 타입은 문자열이고 data의 이름에 "-" 작성 할 수 있다.
  • data-key-key-key : element.dataset.key.key.key 으로 key를 찾을 수 있다.
