POSITION

position 요소, 노드 개체의 위치를 정의한다. (static, relative, absolute, fixed, sticky)

position : static (기본위치 default)

모든 요소, 노드 개체는 위치를 static으로 정의한다.

  • 요소의 배치는 순차적이며(위부터 아래로) 왼쪽에서 부터 오른쪽으로 배치된다.
  • 자식요소는 부모요소의 위치를 기준으로 배치된다.
  • top, bottom, left, right 를 사용 할 수 없다.

position : relative (상대적 위치)

relative BOX

자식 박스를 상대적 위치로 이동

  • 상대적 위치 relative는 정적 위치 static을 기준으로 노드를 이동 시킨다. (top,left,right,bottom)
  • 상대적 위치로 이동한 노드 개체는 다른 노드에 영향을 주지 않는다.

position : absolute (절대적 위치)

absolute BOX
부모 노드가 static 위치 일때.
절대적 위치의 기준은 root 노드인 body이다.
  • absolute는 조상 요소중 relative, absolute, fixed가 정의 되어 있으면 그 조상을 기준으로 위치를 이동 시킨다.
  • 만약 부모, 조상 중에 relative, absolute, fixed가 없다면 body를 기준으로 위치가 조정된다.
  • 부유객체로 다른 요소를 밀거나 당기지 않고 덮어쓰는 형태로 존재한다.
  • absolute로 정의된 노드는 넓이가 content에 맞게 지정된다. 때문에 넓이를 지정하는 것이 좋다.

position : absolute 부모가 relative 일 경우

absolute BOX
부모 노드가 relative 위치 일때.
절대적 위치의 기준은 relative 위치를 가진 부모 노드 기준

position : fixed (고정 위치)

부모가 relative이고 overflow 있는 노드의 fixed 된 제목입니다.

overflow-y : scroll;

overflow-x : hidden;

overflow : hidden;
