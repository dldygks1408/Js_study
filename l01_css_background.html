background-clip : color를 나타내는 영역(default,padding-box,content-box)

background-clip: content-box = 패딩을 제외한 영역에 color를 나타낸다.

background-clip: padding-box = 패딩까지의 영역을 color로 나타낸다.

background-clip: border-box(default) = 테두리까지의 영역을 color로 나타낸다.

background-orign: padding-box(default) = 패딩까지의 영역을 image로 나타낸다.

background-: border-box = 테두리까지의 영역을 image로 나타낸다.

background-: content-box = 내용 영역을 image로 나타낸다.

background-position : (left,right,center,px,% / top,bottom,center,px,%)

background-attachment: fixed = 이미지가 스크롤을 따라 움직인다.
