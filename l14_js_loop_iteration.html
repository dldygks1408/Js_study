자바 스크립트의 루프와 반복

for, forin, forof, while, dowhile 반복문(Iteration Statements)이 있다.

똑같은 명령을 일정 횟수 만큼 반복하기 위해 작성되는 문법 대체로 검사식에 많이 사용된다.

for 반복문

for( 선언문; 조건문; 증가문; ) : for 는 조건이 true 일때 반복실행한다.

    버블 정렬

  • [11,2,20,3,13,1,0,-20] max :
  • [11,2,20,3,13,1,0,-20] 높은 순서대로 정렬하세요.
  • [11,2,20,3,13,1,0,-20] 낮은 순서대로 정렬하세요.

    • 선택 정렬

    전체 중에 가장 높은 값의 위치를 찾아서 앞자리와 바꿔라
    맨 앞자리를 제외하고 제일 높은 값을 찾아서 맨 앞자리로 바로 뒤랑 바꿔라

  • [11,2,20,3,13,1,0,-20] 높은 순서대로 정렬하세요.

forin 반복문

for( key in IterationObject ){ } : key를 가진 객체가 key를 반환 할 수 없을때까지 반복을 실행

JSON{key:value, key:value} => key를 반환, Array[a,b,c] => index를 반환

  • [a, b, c] :
  • {name: "이요한", age=31, email="dldygks@aa.cc"}
  • "abcde"

forof 반복문

for(Object of IterationObject)

반복 가능한 데이터 타입의 객체에서 더이상 내용물이 나오지 않을때까지 반복 실행

{IterationObject}.next() : next()는 포인트가 객체 밖에 있다가 내용물이 있으면 true를 반환 하고 내용물로 포인트를 옮긴다. 그리고 내용물도 반환한다. 만약에 값이 더이상 나오지 않으면 false를 반환한다.

단점은 key 값을 알아내기 힘들다.

배열, Map(json과 비슷), Set(배열인데 복수의 값을 제거)
